TICKETPAIR — Theatre, Cinema & Concert Reservation Website

Ticketpair is a comprehensive, full-stack web application designed for seamless event and cinema ticket management. This project was developed as a collaborative effort by a dedicated team of 5 students from Istanbul Kultur University.

The platform bridges the gap between event organizers and attendees by providing a secure, high-performance, and user-friendly environment for ticket transactions.

KEY FEATURES
👤 User Experience
Live Search Engine: Real-time event discovery using JavaScript Fetch API, allowing users to see instant results without page reloads.

<img width="1880" height="938" alt="Live Search" src="Screenshots/Ekran görüntüsü 2026-03-17 222134.png" />

Dual Theme Support: Persistent Dark & Light Mode toggle that remembers user preference via localStorage.

Integrated Wallet System: A simulated financial ecosystem where users can link credit cards, top up their balance, and purchase tickets using their digital wallet.

<img width="1899" height="816" alt="Wallet System" src="Screenshots/Ekran görüntüsü 2026-03-17 222120.png" />

Multi-Seat Selection: An interactive booking system allowing users to select and manage multiple seats for various events.

Personal Profile Management: A centered, responsive dashboard for managing personal details, saved cards, and purchase history.

<img width="1906" height="940" alt="Profile Management" src="Screenshots/Ekran görüntüsü 2026-03-17 222107.png" />

🛠️ Admin Capabilities
Event Management (CRUD): Full control over event listings, including creating, editing, and deleting cinema or concert entries.

<img width="1907" height="934" alt="Event Management" src="Screenshots/Ekran görüntüsü 2026-03-17 222146.png" />

Dynamic Sales Tracking: A visual analytics tool featuring animated red progress bars that dynamically show ticket saturation and occupancy rates.

Support & QA Center: A dedicated communication hub where admins can respond to user inquiries, edit existing answers inline, or delete tickets.

<img width="1891" height="931" alt="Support Center" src="Screenshots/Ekran görüntüsü 2026-03-17 222209.png" />

💻 Tech Stack

Backend: ASP.NET MVC 5 (.NET Framework 4.8).

Architecture: Model-View-Controller (MVC).

Database & ORM: MSSQL with Entity Framework Code First.

Frontend: HTML5, CSS3 (Custom Styles), JavaScript (ES6+), and Bootstrap.

Asynchronous Logic: AJAX & Fetch API for seamless UI updates.


🏗️ Installation & Setup

Clone the repository:git clone https://github.com/Akiren7/TICKETPAIR-Theatre-Cinema-Concert-Tickets-Reservation-Website.git

Open the solution file (.sln) in Visual Studio 2026.

Update the connection string in Web.config to match your local SQL Server instance.

Run the following command in the Package Manager Console to initialize the database:

Update-Database

Press F5 to build and run the project.
